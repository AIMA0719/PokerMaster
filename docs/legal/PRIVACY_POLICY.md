## 1. 수집·이용하는 개인정보

본 앱은 **어떠한 개인정보도 수집하지 않습니다.**

- 회원가입 없음 (계정 시스템 부재)
- 이름·이메일·전화번호·주소 등 식별정보 미수집
- 위치정보·연락처·통화기록·사진·미디어 미접근
- 광고식별자(ADID) 미수집
- Firebase Analytics, Crashlytics, 기타 분석 SDK 미탑재

## 2. 단말 내부에 저장되는 정보

다음 정보는 **사용자 단말 내부에만 저장**되며, 본 앱 운영자나 제3자에게 전송되지 않습니다.

| 항목 | 위치 | 목적 |
|---|---|---|
| 게임 칩 잔액 | DataStore | 게임 진행 |
| 핸드 히스토리 | Room DB (앱 내부 저장소) | 복기·통계 |
| 닉네임·아바타 선택 | DataStore | 화면 표시 |
| 설정 (애니속도·사운드 등) | DataStore | 사용자 환경 |
| 익명 설치 식별자(UUID) | DataStore | 단말 내 데이터 식별 |

위 정보는 사용자가 [설정 > 데이터 > 전체 초기화] 에서 즉시 삭제할 수 있습니다.
앱 삭제 시 Android 표준에 따라 위 정보 모두 삭제됩니다.

## 3. 모델 다운로드와 외부 통신

본 앱은 온디바이스 AI(LLM) 모델 파일을 최초 1회 또는 업데이트 시 외부 호스팅 사업자로부터 다운로드합니다.

- **호스팅 사업자(우선순위)**: HuggingFace Hub, Cloudflare R2, GitHub Release
- **노출 가능 정보**: 사용자 단말의 IP 주소, User-Agent, 다운로드 시각
- **본 앱 운영자는 어떠한 서버도 운영하지 않으며**, 위 호스팅 사업자에게 남는 로그에 접근하지 않습니다.
- 호스팅 사업자의 개인정보 처리방침은 각 사업자의 공식 페이지를 따릅니다.

사용자가 모델 다운로드를 거부하거나 [설정 > AI > 폴백 모드 강제] 를 활성화하면 위 외부 통신은 발생하지 않으며, 본 앱은 결정형 AI 코어만으로 동작합니다.

## 4. 개인정보 제3자 제공·위탁

본 앱은 개인정보를 수집하지 않으므로 제3자 제공·처리 위탁이 발생하지 않습니다.

## 5. 만 14세 미만 이용자

본 앱은 회원가입이 없으므로 법정대리인 동의 절차가 발생하지 않습니다. 다만 본 앱은 시뮬레이션 도박 모티프(포커 카드게임)를 포함하므로, 만 14세 미만 이용자는 보호자의 지도 하에 사용할 것을 권장합니다. (등급분류는 IARC 기준으로 청소년이용가)

## 6. 이용자 권리

본 앱은 개인정보를 수집·보관하지 않으므로 열람·정정·삭제·처리정지 요청 대상 정보가 부재합니다. 단말 내부 저장 정보는 사용자가 직접 [설정 > 데이터 > 전체 초기화] 또는 앱 삭제로 즉시 제거할 수 있습니다.

## 7. 안전성 확보 조치

- 단말 내부 저장 정보는 안드로이드 표준 앱 샌드박스에 보관됩니다.
- 모델 다운로드는 HTTPS + SHA-256 무결성 검증 + (출시 시) Ed25519 서명 검증.
- 본 앱은 인터넷 권한을 모델 다운로드에만 사용하며, 사용자가 [설정 > AI > 인터넷 권한 사용 안 함] 으로 차단할 수 있습니다.

## 8. 처리방침 변경

본 처리방침이 변경될 경우 앱 내 [설정 > 정보 > 개인정보 처리방침] 화면 및 본 문서 페이지에 사전 공지합니다.
