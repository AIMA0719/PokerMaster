# IARC 등급분류 응답 가이드

> 구글 플레이 콘솔 > 앱 콘텐츠 > IARC 설문 응답 시 사용. 일관된 응답으로 안정적인 등급 확보.
> v1.1 §4.5 IARC 답변 가이드 기반.

---

## 목표 등급

**최종 목표**: 청소년이용가 (12+/15+/Teen) — 보수적으로 **15+ 권장**.

근거:
- 시뮬레이션 도박(포커 카드게임) 포함 → 12+ 욕심 시 IARC 알고리즘이 18+ 자동 상향 위험
- **실제 화폐 베팅·환금·구매 일체 부재** → 18+ 트리거 미충족
- 폭력성·선정성·욕설 0

---

## 카테고리별 응답

| 카테고리 | 응답 | 비고 |
|---|---|---|
| 폭력 | 없음 | 카드게임만 |
| 공포 | 없음 | - |
| 성적 콘텐츠 | 없음 | - |
| 욕설/비속어 | 없음 | NPC 대사 검수 완료 |
| 약물 / 알코올 / 담배 | 없음 | - |
| 시뮬레이션 도박 | **포함** | 포커 카드게임 |
| 실제 화폐 베팅 | **없음** | 환금/거래 일체 없음 |
| 가상 화폐 결제 | **없음** | IAP 부재 (v1) |
| 사용자 간 상호작용 | 없음 | 오프라인 싱글 |
| 사용자 생성 콘텐츠 공유 | 없음 | - |
| 위치 정보 공유 | 없음 | 위치 권한 미사용 |
| 개인정보 공유 | 없음 | 수집 없음 |
| 디지털 구매 | 없음 | - |

## 데이터 안전성 섹션 응답

| 항목 | 응답 |
|---|---|
| 데이터 수집 | **없음** |
| 데이터 공유 | **없음** |
| 보안 관행 | 단말 내 데이터는 안드로이드 표준 샌드박스 보호. 외부 통신은 모델 다운로드(HTTPS + SHA-256 + 서명 검증)만 |
| 데이터 삭제 요청 | 사용자가 [설정 > 데이터 > 전체 초기화] 또는 앱 삭제로 즉시 가능 |

## 카테고리 선택

- 카테고리: **카드(Card)** ← 권장 (글로벌 호환성)
- 회피: 카지노(Casino) — 일부 국가(독일·프랑스·중국 등) 자동 차단

---

## 출시 전 체크

- [ ] IARC 설문에 위 응답으로 일관 입력
- [ ] 데이터 안전성 섹션 "수집/공유 없음" 신고
- [ ] 카테고리 "카드" 선택
- [ ] 처리방침 URL 제출 (GitHub Pages)
- [ ] 등급 부여 결과 확인 (12+/15+ 예상, 18+ 시 즉시 이의 제기)
