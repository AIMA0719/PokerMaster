# 청소년 보호 정책

**시행일자**: 2026-04-23
**서비스명**: 포커마스터(PokerMaster)

## 1. 사행성 부재 선언

본 앱은 다음과 같이 설계되어 청소년의 건전한 성장에 지장을 주는 사행성·환금성 요소를 일체 포함하지 않습니다.

- **현금 결제(IAP) 없음** (v1)
- **게임 머니의 환금·거래·양도 일체 없음**
- **온라인 멀티플레이·실시간 베팅 없음** — 오프라인 AI 대전만
- **광고 SDK 미탑재** (v1)
- 게임 머니가 0이 되어도 일일 보너스로 자동 충전되며, 어떠한 추가 결제도 발생하지 않습니다.

## 2. 이용 시간 보호 기능

청소년의 과몰입 방지를 위해 다음 기능을 제공합니다.

- 누적 플레이 시간 알림 (30분/60분/120분/240분, 디폴트 60분 ON)
- 4시간 누적 시 강제 휴식 다이얼로그 (스누즈 1회 후 30분 후 강제 표시)
- 야간 시간대(22~07시) 차단 옵션

## 3. 미성년자 사용 안내

본 앱은 회원가입 없이 누구나 설치·사용할 수 있으나, 다음 사항을 권장합니다.

- 만 14세 미만 이용자는 보호자의 지도 하에 사용
- 본 앱은 시뮬레이션 도박 모티프(포커 카드게임)를 포함하며, IARC 등급분류 기준으로 청소년이용가 등급에 해당합니다
- 보호자는 단말의 디지털 웰빙 / 패밀리 링크 기능으로 사용 시간을 추가 관리할 수 있습니다

## 4. 청소년 유해 콘텐츠 부재

본 앱은 다음 요소를 포함하지 않습니다.

- 폭력성: 없음
- 선정성: 없음
- 욕설/비속어: NPC 대사는 가족 친화 톤으로 검수
- 약물·알코올·담배: 없음
- 공포·혐오: 없음

## 5. 향후 개선

본 정책은 향후 v2 광고/IAP 도입 시 다음과 같이 강화됩니다.

- 광고 도입 시: 청소년 광고 필터 강제 (`maxAdContentRating='T'`), 도박/주류/성인 카테고리 차단
- IAP 도입 시: 게임 머니 직접 판매 금지, 콘텐츠(스킨/아바타) 판매로 한정
