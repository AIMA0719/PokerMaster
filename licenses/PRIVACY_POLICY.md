# PokerMaster 개인정보 처리방침

최종 업데이트: 2026-04-28

## 1. 수집 항목

PokerMaster 는 사용자 식별 가능 개인정보를 **수집하지 않습니다**.

다음 정보만 사용자 단말 로컬에 저장됩니다:
- **게임 진행 데이터**: 핸드 히스토리 (액션 로그·시드·결과), 칩 잔액, 일일 streak,
  설정 (효과음·햅틱·접근성·테마).
- **모델 캐시**: 로컬에 저장된 LLM 모델 파일 (`.gguf`) 의 무결성 검증용
  SHA-256 해시.

## 2. 외부 전송

- 외부 서버로 전송되는 사용자 데이터는 **없습니다**.
- 유일한 외부 통신: 최초 1회 LLM 모델 다운로드 (HuggingFace public CDN).
  - 전송 정보: HTTP User-Agent (`PokerMaster/{버전} (Android)`) 외 PII 없음.
  - 다운로드 실패/재시도 시에도 사용자 식별 정보 미포함.

## 3. 광고 / 분석 / 트래킹

- 광고 SDK **사용 안 함**.
- 분석 SDK (Firebase Analytics 등) **사용 안 함**.
- 사용자 행동 트래킹 **하지 않음**.
- 광고 식별자 (AAID) **수집 안 함**.

## 4. 권한 사용 목적

| 권한 | 목적 |
|---|---|
| INTERNET | LLM 모델 최초 1회 다운로드 |
| ACCESS_NETWORK_STATE | Wi-Fi/셀룰러 상태 확인 (다운로드 정책) |
| POST_NOTIFICATIONS | 다운로드 진행/완료 알림 (Android 13+) |
| FOREGROUND_SERVICE | 백그라운드 다운로드 유지 |

## 5. 데이터 보존 / 삭제

- 모든 데이터는 사용자 단말 내부 저장소 (`/data/data/.../`) 에 저장.
- 앱 설정 → 데이터 → "히스토리 전체 삭제" 로 즉시 삭제 가능.
- 앱 삭제 시 모든 데이터 함께 제거 (`allowBackup=false` 적용).

## 6. 어린이 보호

- 만 14세 미만 사용자를 대상으로 하지 않습니다.
- 본 앱의 칩은 가상 점수이며 실제 환금/거래/결제가 불가능합니다.
