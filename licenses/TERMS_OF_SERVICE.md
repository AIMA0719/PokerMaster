# PokerMaster 이용약관

최종 업데이트: 2026-04-28

## 1. 서비스 정의

PokerMaster ("본 앱") 은 오프라인 싱글플레이 포커 시뮬레이션 앱입니다.
- 한국식 7카드 스터드 / 7카드 스터드 하이로우 / 텍사스 홀덤 모드 제공.
- 온디바이스 AI (LLM) 가 NPC 행동 결정.
- 칩은 가상 점수 — 실제 환금/거래/결제 불가.

## 2. 이용 자격

- 만 14세 이상 사용자.
- 본 앱은 도박이 아니며, 칩은 환금성이 없는 가상 점수입니다.

## 3. 사용자 의무

- 본 앱을 다음 목적으로 사용 금지:
  - 실제 금전이 오가는 도박 행위 매개.
  - 자동화 봇/매크로/리버스 엔지니어링 (오프라인 싱글플레이라 의미 없으나 명시).
  - 본 앱의 코드/모델 가중치 재배포 (오픈소스 라이선스 별도 — Apache/MIT/Llama
    Community License 참조).

## 4. 책임 제한

- 본 앱은 "있는 그대로 (AS IS)" 제공됩니다.
- AI NPC 의 결정은 학습 모델의 한계로 비논리적일 수 있으며, 이는 정상
  동작입니다.
- 모델 다운로드 실패/단말 호환성/배터리 소모는 사용자 환경 의존.

## 5. 서비스 변경 / 종료

- 향후 업데이트로 게임 모드 / 모델 / UI 가 변경될 수 있습니다.
- 서비스 종료 시 앱 업데이트로 안내합니다.

## 6. 분쟁 해결

본 약관은 대한민국 법률을 따릅니다.
